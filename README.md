# Suivi Voyage Madagascar

Application web statique mobile-friendly pensée pour un usage terrain avec connexion limitée.

## Objectifs

- consultation rapide sur mobile
- navigation évidente
- données robustes même hors ligne partiel
- stockage local pour les éléments modifiables

## Structure

- `index.html` : accueil et tableau de bord
- `pages/` : pages fonctionnelles
- `assets/css/style.css` : styles communs
- `assets/js/` : logique par page + stockage
- `assets/data/` : données de référence au format JSON

## Stratégie de données

- HTML : structure de page
- JSON : données préparées à l'avance
- localStorage : dépenses, notes, taux mémorisé
